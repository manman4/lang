# PARI



| 命令 | 操作 |
| :--: | :--: |
| \r ファイル.gp | gpファイルの実行 |
| default(parisize, 120000000) | parisizeの変更 |
| \p 56 | 実数の精度が 56 桁になる. 実数のデフォルトの 10 進での桁数は、28 |


# 基本演算

lift であまりを取り出す。

```PARI:
? x = Mod(3, 5)+Mod(4, 5)                                                                     
%1 = Mod(2, 5)
? lift(x)
%2 = 2
```

max, min

```PARI:
? max(3, 5)
%1 = 5
? min(3, 5)
%2 = 3
```

約数のベクトル

```PARI:
? v = divisors(24) 
%1 = [1, 2, 3, 4, 6, 8, 12, 24]
? for(k=1, #v, print1(v[k],", "));
1, 2, 3, 4, 6, 8, 12, 24, 
```

ベクトルから形式的冪級数

```PARI:
? A = [1, 2, 3, 4];
? Ser(A)
%2 = 1 + 2*x + 3*x^2 + 4*x^3 + O(x^4)
? Ser(A)^2
%3 = 1 + 4*x + 10*x^2 + 20*x^3 + O(x^4)
```
