# 🚀 lang ✨

言語の比較等行う。

自分用辞書

環境構築は「environment_setup」でまとめている。

## 対象言語

Go, PARI, Python, Ruby, Java

--------------------

文字列の操作

| 操作 | Go | Ruby |
| :--: | :--: | :--: |
| 数字へ | n, _ = strconv.Atoi(s) | n = s.to_i |
| 文字へ | s = strconv.Itoa(n) | i = n.to_s |
